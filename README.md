MERN Product Store

A simple full-stack product management system built with the MERN stack.
Users can create, view, update, and delete products with image upload support.

Features
Create new products (name, price, image)
Upload product images
View all products in a responsive grid layout
Edit products using a modal popup
Delete products instantly
Real-time UI updates (no page refresh)
RESTful API backend with Express & MongoDB

Frontend:
React
Zustand (state management)
Tailwind CSS
React Router

Backend:
Node.js
Express.js
MongoDB + Mongoose
Multer (file upload)
